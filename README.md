# dotfiles
My dotfiles for assorted apps and wms

Included dotfiles for:
*i3wm (i3-gaps if you uncomment the gaps section)

*qutebrowser

*alacritty

*ranger

*polybar

# How to install:
 `cd ~/.config`
 
 `git clone https://github.com/konkywill/dotfiles.git `
 
 `mv ~/.config/.alacritty.yml ~/`
